In accordance with the Personal Information Protection Law, DimArea Games (hereinafter referred to as "the company") has the following policy to protect the privacy of users and protect their rights.


1. Purpose of processing personal information

The company only collects essential information to take advantage of the game center's functions.
Essential information includes account information to access the Game Center, and game data that contains progress of the game.


2. How to handle personal information

A) Access to the Game Center

The company uses only external plug-ins for each operating system when the user accesses the game center.
In other words, when the user accesses the game center, the company does not obtain any personal information by using other means nor store it.
For Android apps, official GPGS for Unity plugin provided by Google is used.
For iOS apps, iOS Native Pro plug-in provided by Stan's Asset is used.

B) Game Data

The company stores the user's game data in the official remote storage provided by each operating system developer.
Android apps store data in Google Cloud, and iOS apps store data in iCloud.
The way to store this game data is done using the above plug-in.
Game data is the data that contains the progress of the game.
The company stores it in remote storage and uses it to synchronize users' local data with cloud data.
The company also uses it to provide leaderboard and achievement capabilities.
Leaderboards and achievements are also provided using the above plug-in.


3. Storing personal information

The company handles all personal information of users with external plugins only.
In other words, the company does not store any personal information of users in a separate repository.
Therefore, there is no period of personal information retention.


4. Third-party offers of personal information

The company does not provide personal information to third parties.


5. Entrustment of personal information

The company does not entrust personal information processing.


6. Rights of users

1) If there is an error, you can request correction.
2) You can request the deletion of personal information.


7. Modifying the Privacy Policy

If the company modifies the privacy policy, it will upload the updated processing policy to the company's official website.
